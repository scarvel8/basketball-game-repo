# Basketball Game

This is a Game Boy game created using GB Studio.

## Project Structure
- `basketball.gb`: The compiled Game Boy ROM.
- `basketball-game.gbsproj`: The GB Studio project file.
- `assets/`: Contains game assets like backgrounds, sprites, and fonts.
- `project/`: Contains GB Studio project data like scenes, actors, and scripts.

## How to Play
To play the game, load `basketball.gb` into a Game Boy emulator or flash it onto a compatible cartridge.

## Development

### Building
To build this project, open `basketball-game.gbsproj` in GB Studio and use the "Build and Run" or "Export ROM" options.
